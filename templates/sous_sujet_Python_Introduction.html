Variables

Variables are containers for storing data values.
There is no command to declare a variable so it is created
at the moment you first assign a value to it. It does not need a type
and it can change type after they have been set.
Finally it is possible to specify the type of a variable with a casting


Exemple :

x = 3 # x is type int
x = "test" # x is now str
y = str(4) # y will be '4'
z = float(4) # y will be 4.0



Data Types

  • Text Type: str

  • Numeric Types: int, float, complex

  • Sequence Types: list, tuple, range

  • Mapping Type: dict

  • Set Types: set, frozenset

  • Boolean Type: : bool

  • Binary Types: bytes, bytearray, memoryview

  • None Type: NoneType


Exemple :

In python the data type is set when you assign a value to a variable:

x = 3456 # x is type int
y = "Hello World" # y is a str
z = ["apple","banana","orange"]# z is a list

Python numbers


There are three types of numbers in Python.
- Int
- Floats
- Complex

Exemple :

The numbers can have positive and negative values.

x = -34 #x is an int
y = 12.3 #y is a float
z = 1j #z is a complex
w = -5j #w is a negative complex
