Python

Introduction

Python est un langage de programmation populaire, créé par Guido van Rossum, en 1991.

Capacités de Python

Il est notamment utilisé pour le développement côté serveur (back-end), le développement de logiciel, les mathématiques ainsi que les scripts système.

Capacités de Python

Il peut être utilisé pour créer des applications web, pour connecter des systèmes de base de données ainsi que lire ou modifier des fichiers ainsi que pour gérer des big data et résoudre des mathématiques complexes, entre autres.

Comment justifier son usage?

Python est une alternative intéressante puisqu'il fonctionne sous plusieurs plateformes (Windows, Mac, Linux, Raspberry Pi, etc.). Il utilise une syntaxe similaire à l'anglais, rendant l'usage très intuitif. Il peut également être traité de manière précdurale, orientée-objet ou de manière fonctionnel.

Les différences au niveau de la syntaxe

Il a été concu pour sa grande facilité à lire, vu sa proximité avec l'anglais tout en se laissant influencer par les mathématiques. Python utilises des nouvelles lignes pour compléter une commande, comparativement à d'autres langages qui utilises des points-virgules ou des parenthèses. Il se base sur l'indentation et l'usage de whitespaces. D'autres langages de programmation utilises souvent à l'instar les accolades.
