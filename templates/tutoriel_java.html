Comments


Comments can be used for differents purposes :

  • Explain Java code.
  • Make the code more readable.
  • Prevent execution when testing code.


Example


// This line is a comment

Variables and Data types


In Java, there are different types of variables, for example:
String - stores text, such as "Programmation". String values are surrounded by double quotes.
int - stores integrers (whole numbers), without decimals, such as 49 or -49
float - stores floating point numbers, with decimals, such as 2.33 or -2.33
char - stores singles characters, such as 'a' or 'A'. Char values are surrounded by single quotes
boolean - stores values with two states: true or false


Example :


int x = 77 # x is an int
String animal = "dog" # animal is a str
char y = 'y' # y is a char
# y will be '4'
z = float(4) # y will be 4.0
