Variable Names

The rules to define a variables in python:

  • It must start with a letter or the underscore character
  • It cannot start with a number
  • It only contain alpha-numeric and underscore characters
  • Finally it is case sensitive

Exemple

myvar = "John"
my_var = "John"
_my_var = "John"

Multiple value assignement on one line

  • In python, it is possible to assign values to multiple variables in one line
  • You can also assign the same value to multiple variables in one line
  • If you have a collection of values, you can extract them into variables

Exemple

x, y, z = "Orange", "Banana", "Cherry"
x = y = z = "Orange"
fruits = ["apple", "banana", "cherry"]

Output Variables

  • The Python print() function is often used to output variables.
  • If you output multiple variables you can use a coma to separate them
  • you can also use the + operator
  • BUT for number, the operator + work as mathematical operator
  • Finally if you mix number and String with the
    operator + you will have an error

Global Variables

Variables that are created outside of a function are known as global variables.

  • Global variables can be used from inside and outside of a function
  • If you create a variable with the same name inside a function, this variable will be local, and can only be used inside the function.
  • To change the value of a global variable inside a function, refer to the variable by using the global keyword

The numbers can have positive and negative values.

Exemple

x = "awesome" #global variables declaration
def myfunc() :
global x #Refer to the global variable x
x = "fantastic" #reassign "fantastic" to the global variable
myfunc()
print("Python is" + x ) #the output is "Python is fantastic"
